# Meeting Scraper Desktop App

A native Mac app wrapper for Meeting Scraper.

## Quick Start

### Run in development mode:
```bash
cd desktop
npm install
npm start
```

### Build the Mac app:
```bash
cd desktop
npm install
npm run build
```

The built `.dmg` file will be in `desktop/dist/`.

## Requirements

- Node.js 18+
- macOS (for building Mac apps)

## Notes

- The app loads the production site (https://meeting-scraper-frontend.onrender.com)
- External links open in your default browser
- Use Cmd+R to refresh
