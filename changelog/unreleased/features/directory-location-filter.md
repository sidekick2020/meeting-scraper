**Directory Location Filter with Radius**: Added location-based filtering to the directory
- Smart location search with autocomplete using OpenStreetMap/Nominatim
- Configurable radius filter (5, 10, 25, 50, 100 miles)
- Distance column appears when location is set, showing distance to each meeting
- All columns are now sortable (ascending/descending) by clicking on headers
- Meetings are sorted by distance by default when location filter is active
