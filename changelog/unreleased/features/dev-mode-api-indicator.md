**Dev Mode API Indicator**: Added an expandable API request logger for development environments
- Automatically detects development mode based on hostname (enabled on Render preview URLs, disabled on production)
- Shows all API requests in real-time with method, URL, status code, and duration
- Expandable panel in bottom-right corner with request/response details
- Copy requests as JSON or cURL commands for easy debugging
- Helps developers debug API issues faster during development
