**User Permissions System**: Granular permission management for dashboard users
- Expandable user cards in Settings to view and modify individual permissions
- Seven permission categories: View Meetings, Manage Meetings, Run Scraper, View Heatmap, View Analytics, Manage Tasks, Manage Users
- Admin users automatically receive all permissions
- Domain users (sobersidekick.com, empathyhealthtech.com) who join without invite get view-only access by default
- Permission toggles with category grouping for easy management
