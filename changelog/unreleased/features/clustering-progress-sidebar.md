**Clustering Progress Sidebar**: Real-time progress visualization for Map Indicator Job
- Slide-out sidebar shows detailed progress when clustering job is running
- ETA calculation based on elapsed time and current progress percentage
- Phase progress indicators showing current operation (fetching, clustering, saving, etc.)
- Real-time log streaming with auto-scroll to latest entries
- Statistics display showing meetings processed, indicators created
- Mini progress bar in the "Show Progress" button
- Auto-opens when starting a new clustering job
- Supports both Full Rebuild and Incremental modes
